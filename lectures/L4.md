# Lecture 4

## FSM

Finite Number of states

Finite number of external inputs and outputs

### How to build FSM

We need state register, next state logic and output logic

State register is sequential circuit. It stores the current state.

State register is made using D Flip FLops.

Moore FSM: Output only depends on current state.

Mealy FSM: Output depends on current state and input.

### FSM Design Procedure

Develop a state transition diagram

Start by defining the reset state

Continue adding transitions and states
